# Synapse Homeserver — Docker Compose Setup

This repository provides a single Docker Compose file that sets up a complete Matrix homeserver environment, including:

## Services Included
- Synapse Homeserver
- LiveKit Server for Element Call
- Sygnal (Push Gateway Service)
- Email integration for Synapse  
  - Requires an external SMTP server
- Caddy (Reverse proxy — configured externally)
